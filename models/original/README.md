# Modele bazowe

Ten katalog jest przeznaczony na pliki z projektu `ln-komandur/elna-cam-discs`.

Import automatyczny:

Linux/macOS/Git Bash:
```bash
bash tools/import_source_models.sh
```

Windows PowerShell:
```powershell
powershell -ExecutionPolicy Bypass -File tools/import_source_models.ps1
```

Po imporcie powinny pojawić się m.in.:
- `Cam-as-18-Sided-Polygon.scad`
- `Single Cam - No 13.scad`
- `Single Cam - No 16.scad`
- `Single Cam - No 20.scad`
- `Single Cam - No 33.scad`
- `Double Cam - No 107.scad`
- `font.dxf`
