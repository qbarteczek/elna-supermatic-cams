# Elna Supermatic Cam Library

Otwarta biblioteka profili krzywek/dysków do maszyn Elna Supermatic, Elna SU i modeli pokrewnych.

## Cel
Odtworzenie pełnego zestawu dysków na podstawie istniejących modeli OpenSCAD, pomiarów oryginałów i zdjęć referencyjnych.

## Struktura
- `models/original/` – modele bazowe 13, 16, 20, 33 i 107 po skopiowaniu z projektu źródłowego,
- `models/generated/` – nowe modele,
- `references/` – zdjęcia, linki i opisy źródeł,
- `tools/` – narzędzia do ekstrakcji konturu,
- `docs/` – dokumentacja i indeks dysków.
