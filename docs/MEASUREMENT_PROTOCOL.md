# Measurement and photo protocol

To improve or verify a cam profile, provide:

## Best case

- top-down photo of the cam,
- bottom photo,
- visible cam number,
- visible stitch pictogram,
- no strong perspective,
- known scale, e.g. caliper or ruler,
- ideally 1500 px or more across the disc.

## Minimum useful photo

- full outer cam edge visible,
- central hole visible,
- locating/transport slot visible,
- cam number visible.

## Verification checklist

A model can be marked as verified only after:

1. profile extracted or measured from a real cam,
2. OpenSCAD model compared visually,
3. printed test part fits without force,
4. machine rotates by hand safely,
5. stitch output matches the expected pattern.
