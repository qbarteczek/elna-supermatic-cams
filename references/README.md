# Materiały referencyjne

Dla każdego dysku zapisz:
- numer,
- adres źródła,
- datę dostępu,
- orientację,
- wymiar odniesienia,
- jakość zdjęcia,
- informację, czy zdjęcie wykonano prostopadle.

Nie kopiuj cudzych zdjęć bez sprawdzenia licencji. Bezpieczniej przechowywać link, miniaturę roboczą i własny obrys.
