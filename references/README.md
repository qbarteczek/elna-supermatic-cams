# Materiały referencyjne

Dla każdego dysku twórz osobny plik:

```text
references/cam_XX_sources.md
```

Nie wrzucaj cudzych zdjęć bez licencji. Bezpieczniej zapisać:
- link,
- datę dostępu,
- opis zdjęcia,
- ocenę jakości,
- własny profil/obrys.

## Szablon źródła

```markdown
# Cam XX — źródła

## Źródło 1

- URL:
- Data dostępu:
- Numer dysku widoczny: tak/nie
- Widok prostopadły: tak/nie/częściowo
- Otwór transportowy widoczny: tak/nie
- Jakość: dobra/średnia/słaba
- Uwagi:
```
